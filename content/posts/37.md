> 减少每次代码执行时的重复性的分支判断，通过对对象重定义来屏蔽原对象中的分支判断

### 对事件的思考

```js
// 单例模式定义命名空间
var A = {};
A.on = function (dom, type, fn) {
  if (dom.addEventListener) {
    dom.addEventListener(type, fn, false);
  } else if (dom.attachEvent) {
    dom.attachEvent("on" + type, fn);
  } else {
    dom['on' + type] = fn;
  }
};
```
